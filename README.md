
# Furniture Finder

Furniture Finder is a modern web application that allows users to search for furniture and view the top 10 highest-rated items based on web-scraped data.

## Features
- Furniture search
- Web scraping via Node.js API
- Top 10 highest-rated items
- Photos, prices, descriptions, and direct links
- Deployed easily on Vercel

## Technologies Used
- Next.js (React + Node.js)
- Axios
- Cheerio (Web Scraping)
- Vercel (Deployment)

## Deployment Steps

1. Unzip the project
2. Push to GitHub
3. Go to https://vercel.com
4. Import GitHub repository
5. Click Deploy
6. Visit the generated URL

## Local Development
```bash
npm install
npm run dev
```
Visit http://localhost:3000
